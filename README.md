# 國中自然 5-1 ~ 5-2 線上測驗系統 🧪

這是一個針對國中八年級自然科（理化）第五章「溫度與熱」設計的線上互動測驗網頁。

## 專案簡介
為了讓學生能夠在課後進行自我評量，我設計了這個不需要登入、開啟即用的網頁程式。內容涵蓋 5-1 溫度計原理與 5-2 比熱與熱平衡。

## 🚀 線上試用 (Demo)
點擊以下連結即可開始測驗：
[👉 點此進入測驗網頁](https://您的帳號.github.io/science-quiz/)
*(請將上面的連結換成您實際的 GitHub Pages 網址)*

## ✨ 功能特色
* **即時回饋**：作答後立即顯示對錯。
* **觀念詳解**：針對錯誤選項提供導正觀念的解釋（像老師在旁解說一樣）。
* **響應式設計**：在電腦、平板或手機上都能順暢操作。
* **數學公式支援**：使用 MathJax 顯示比熱公式 $H = M \times S \times \Delta T$。

## 🛠️ 使用技術
* HTML5
* CSS3 (Flexbox layout)
* Vanilla JavaScript (純 JS，無依賴框架)

## 題目來源
改編自 114學年度國中自然2上備課用書 (第五章 5-1, 5-2)。
